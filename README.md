# pyChat

A Simple chatroom server and client written in python.

### Prereq's:

- Python 2.7

### Usage:

- Modify the files to change the ip's and ports
- run the server:
    - Supports flags
    -     python server.py
    -     python server.py localhost 6800
    -     python server.py localhost:6800
- run the client in a seperate terminal
    - Supports flags
    -     python client.py
    -     python client.py localhost 6800
    -     python client.py localhost:6800
